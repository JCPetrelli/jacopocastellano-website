## About

A collection of bash scripts designed for performing various types of backups. The toolkit includes scripts for incremental backups, full system backups, and selective directory synchronization.

## Features

- Incremental backup support
- Configurable retention policies
- Logging and notification system
- Cross-platform compatibility (macOS/Linux)
