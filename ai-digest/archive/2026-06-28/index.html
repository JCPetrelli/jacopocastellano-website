    Asian startups are rushing out Mythos-like models that carry no export ban, with some warning U.S. labs may permanently lose that market, while Europe pushes harder for sovereign models of its own.   3. The scramble to escape Nvidia is accelerating.

    OpenAI detailed Jalapeño, a custom inference chip built with Broadcom, joining Google, Apple, and SpaceX in designing their own silicon to cut single-supplier risk. China added a twist by claiming the world's fastest supercomputer, LineShine, built without any GPUs.

    Sources TechCrunch ·Wired

  4. Physical AI is moving from labs to factory floors.

    AGIBOT shipped its 15,000th robot and General Intuition raised $320M to train robots on video-game footage, while Flexiv, Kawasaki, and ABB showed off new dexterous, force-controlled "physical AI" arms at Automate 2026. Funding is flowing into the supporting stack too, with Limitless Labs raising $20M for AI-driven CNC programming.

    Sources The Robot Report ·The Robot Report ·Robotics & Automation News ·Robotics & Automation News

  5. AI coding assistants are becoming an attack surface.

    Researchers showed an Amazon Q flaw that let booby-trapped Git repositories execute code and steal cloud credentials, warning that many assistants now run commands straight from project config files. More broadly, security teams face a messy summer as AI tools surface countless previously hidden vulnerabilities.

    Sources The Register ·The Register

  6. Tooling for autonomous agents is maturing fast.

    Vercel open-sourced Eve, a filesystem-based framework for building and running production agents, and AWS previewed a FinOps Agent that investigates cost anomalies and routes findings to Slack and Jira. The shift is real enough that Notion is shutting its Gmail client because more than half of users now let bots handle email.
